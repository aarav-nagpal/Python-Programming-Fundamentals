# Secret Message Encryptor

A Python CLI tool that encrypts and decrypts text files using a custom logic algorithm.

### Features
- **Custom Encryption:** Shifts strings and injects random "noise" characters to mask data.
- **File Handling:** safely reads/writes to `coded.txt` and `decoded.txt`.
- **Error Handling:** Manages missing files and invalid user inputs to prevent crashes.

### Usage
Run the script in a terminal:
python cipher_tool.py
